# 📝 데이터 확인 팁 (ChatGPT Agent가 놓칠 수 있는 부분)

## 네이버 금융 페이지 구조

### 1. 기본정보 위치
- **현재가**: 페이지 최상단 가장 큰 숫자
- **전일대비**: 현재가 바로 아래 (빨강/파랑 화살표)
- **거래량**: 현재가 우측 박스

### 2. 투자지표 위치
- 우측 "투자정보" 박스
- PER, PBR, ROE, 배당수익률 모두 표시됨

### 3. 실적 컨센서스 위치
1. "종목분석" 탭 클릭
2. 페이지 스크롤 내림
3. "투자의견" 또는 "컨센서스" 섹션
4. 최신 분기(4분기) 데이터 확인

### 4. 증권사 목표가 위치
1. "종목분석" 탭
2. "증권사 리서치" 또는 "투자의견" 섹션
3. 최근 날짜순 정렬
4. 상위 3개 선택

### 5. 수급 데이터 위치
1. "투자자별 매매동향" 탭 클릭
2. 일별/주별/월별 탭 전환
3. 외국인/기관/개인 순매수/순매도 금액

### 6. 뉴스 위치
1. "뉴스/공시" 탭
2. 최근순 정렬
3. 중요 뉴스 선별
